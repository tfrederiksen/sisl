class sisl.Orbital(R, q0=0.0, tag='')ÔÉÅ

Bases: object

Base class for orbital information.

The orbital class is still in an experimental stage and will probably evolve over some time.

Parameters

  • R (float) ‚Äì maximum radius

  • q0 (float, optional) ‚Äì initial charge

  • tag (str, optional) ‚Äì user defined tag

Examples

>>> orb = Orbital(1)
>>> orb_tag = Orbital(2, tag="range=2")
>>> orb.R == orb_tag.R / 2
True
>>> orbq = Orbital(2, 1)
>>> orbq.q0
1.
