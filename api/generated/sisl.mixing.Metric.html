sisl.mixing.Metric

class sisl.mixing.Metric(metric=None)

Bases: object

Perform inner products using a metric

An inner product can be defined as:

\[s = \langle \mathbf a | \mathbf M | \mathbf b \rangle\]

where generally the metric \(\mathbf M = 1\).
